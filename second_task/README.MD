# Задача на подход к программированию
```
ТЗ: необходимо написать базовый скрипт, реализующий следующий сценарий: 
на город нападает антагонист, жители просят супергероя спасти город 
и позволяют ему решить проблему в собственном стиле. 
После неизбежной победы добра над злом в СМИ появляется оповещение о радостной новости.

Важные детали: 
К определенному городу привязан свой определенный пулл антагонистов 
(к Костроме привязан Годзилла, например)
У каждого супергероя свои уникальные способы атаки (лазер, оружие или все вместе)
Каналов СМИ может быть несколько (газета, tv и тд)
В качестве входных параметров выступают: супергерой, 
город и канал СМИ. Например: (ЧакНоррис, Токио, TV)
Настоятельно рекомендуется использование принципов SOLID
```
Результат выполнения скрипта может выглядеть так:
```
Godzilla stands near a skyscraper
PIU PIU
Today in newspapers: Chuck Norris saved Tokyo!
--------------------
Aliens abduct cows in the meadow
PIU PIU
Watch on the first channel: Chuck Norris saved the Kostroma!
```
## Result
Start `main.py` file, пример результата:
```
Today 2024-02-17 17:07:41 - <Mai Shen>
Stands near a skyscraper in London and attack with magic
After PIU PIU of <Mauler> with magic, city was saved
Watch today on TV - Tokyo News
```
